Instruction on how to run the code:
Codes can be accessed via this link: https://github.com/cliffrunner/Machine_Learning/tree/master/HW1

1. Download HW1.ipynb, eeg_dataset.csv, otto_dataset.csv to a folder
2. Install packages to run the code, which are specified in the '1.1 Requirements' section in ypeng8-analysis.pdf
3. Open HW1.ipynb from Jupyter Notebook
4. In each cell, press ctrl+enter to run each cell; or click the 'Run' button from top to run all cells in sequence (not recommended because it takes forever)
